### Installation

React-Virgin requires [Node.js](https://nodejs.org/) v4+ to run.

Clone the repo

```sh
$ cd Trixie-UI-kit
$ cd TrixieUiKit
$ npm install
$ react-native run-ios
```

Linting
```
$ yarn run eslint
```
Use linter plugins for eslint on Sublime/Atom/VSCode
